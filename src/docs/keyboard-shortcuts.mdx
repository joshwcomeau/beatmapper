Beatmapper makes heavy use of keyboard (and mouse) shortcuts. You can make maps without learning any of them, but your life will be much easier if you take the time to familiarize yourself with them :)

## General shortcuts

These shortcuts can be used either in the "Notes" view or the "Events" view.

## Notes view shortcuts

These shortcuts are exclusive to placing blocks, mines, and obstacles in the "Notes" view.

<ShortcutTable>
  <Shortcut
    keys={
      <>
        <IconRow>
          <KeyIcon>W</KeyIcon>
        </IconRow>
        <IconRow>
          <KeyIcon>A</KeyIcon>
          <KeyIcon>S</KeyIcon>
          <KeyIcon>D</KeyIcon>
        </IconRow>
      </>
    }
  >
    Set the direction for note blocks.
    <Sidenote>
      Diagonals can be achieved by holding multiple keys (eg.{' '}
      <InlineIcons>
        <KeyIcon size="small">W</KeyIcon>
        <KeyIcon size="small">A</KeyIcon>
      </InlineIcons>{' '}
      for up-left).
    </Sidenote>
  </Shortcut>
  <Shortcut
    keys={
      <>
        <IconRow>
          <KeyIcon>F</KeyIcon>
        </IconRow>
      </>
    }
  >
    Set the note direction to a "face" block
    <Sidenote>
      While{' '}
      <InlineIcons>
        <KeyIcon size="small">W</KeyIcon>
        <KeyIcon size="small">A</KeyIcon>
        <KeyIcon size="small">S</KeyIcon>
        <KeyIcon size="small">D</KeyIcon>
      </InlineIcons>{' '}
      choose cardinal directions, <KeyIcon size="small">F</KeyIcon> resets to a
      directionless block.
    </Sidenote>
  </Shortcut>
  {/*  */}
  <Shortcut
    keys={
      <IconRow>
        <KeyIcon>Shift</KeyIcon>
        <Plus />
        <Mouse />
      </IconRow>
    }
  >
    Look around
    <Sidenote>
      Holding shift allows you to pan the camera, like looking around in a
      first-person video game.
    </Sidenote>
  </Shortcut>
  <Shortcut
    keys={
      <>
        <IconRow>
          <KeyIcon>Shift</KeyIcon>
          <Plus />
        </IconRow>
        <IconRow>
          <KeyIcon>W</KeyIcon>
          <KeyIcon>A</KeyIcon>
          <KeyIcon>S</KeyIcon>
          <KeyIcon>D</KeyIcon>
        </IconRow>
      </>
    }
  >
    Move around
    <Sidenote>
      Like Unreal Tournament!
      <br />
      You can also rise straight up with{' '}
      <InlineIcons>
        <KeyIcon size="small">R</KeyIcon>
      </InlineIcons>
      , or fall straight down with <InlineIcons>
        <KeyIcon size="small">F</KeyIcon>
      </InlineIcons>
    </Sidenote>
  </Shortcut>
  {/*  */}
  <Shortcut
    keys={
      <>
        <IconRow>
          <KeyIcon>Backspace</KeyIcon>
        </IconRow>
      </>
    }
  >
    Reset view
    <Sidenote>Jump back to the original camera position</Sidenote>
  </Shortcut>
  {/*  */}
  <Shortcut
    keys={
      <>
        <IconRow>
          <KeyIcon>R</KeyIcon>

          <KeyIcon>1</KeyIcon>
        </IconRow>
      </>
    }
  >
    Set tool to Red block
    <Sidenote>
      So that clicking on the placement grid creates red blocks
