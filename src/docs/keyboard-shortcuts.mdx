Beatmapper makes heavy use of keyboard (and mouse) shortcuts. You can make maps without learning any of them, but your life will be much easier if you take the time to familiarize yourself with them :)

## General shortcuts

These shortcuts can be used either in the "Notes" view or the "Events" view.

## Notes view shortcuts

These shortcuts are exclusive to placing blocks, mines, and obstacles in the "Notes" view.

<ShortcutTable>
  <Shortcut
    keys={
      <>
        <IconRow>
          <KeyIcon>W</KeyIcon>
        </IconRow>
        <IconRow>
          <KeyIcon>A</KeyIcon>
          <KeyIcon>S</KeyIcon>
          <KeyIcon>D</KeyIcon>
        </IconRow>
      </>
    }
  >
    Set the direction for note blocks.
    <Sidenote>
      Diagonals can be achieved by holding multiple keys (eg.{' '}
      <InlineIcons>
        <KeyIcon size="small">W</KeyIcon>
        <KeyIcon size="small">A</KeyIcon>
      </InlineIcons>{' '}
      for up-left).
      <br />
      For face blocks, you can press{' '}
      <InlineIcons>
        <KeyIcon size="small">F</KeyIcon>
      </InlineIcons>
      .
    </Sidenote>
  </Shortcut>
  <Shortcut
    keys={
      <IconRow>
        <KeyIcon>Shift</KeyIcon>
        <Plus />
        <Mouse />
      </IconRow>
    }
  >
    THe mouse and the letter A do something
  </Shortcut>
  <Shortcut
    keys={
      <IconRow>
        <Mouse />+<KeyIcon>A</KeyIcon>
      </IconRow>
    }
  >
    THe mouse and the letter A do something
  </Shortcut>
  <Shortcut
    keys={
      <IconRow>
        <Mouse />+<KeyIcon>A</KeyIcon>
      </IconRow>
    }
  >
    THe mouse and the letter A do something
  </Shortcut>
</ShortcutTable>
